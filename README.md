# clinic
Web application for a clinic managing patients', doctors' and appointments' data based on microservices architecture.

# Tech stack
### Backend
- Java 11
- Spring Boot 2.3.9
- Spring Cloud Netflix
- Spring Data
- Spring Security
- PostgreSQL
### Frontend
- Vue.js 3.0.11
- Bootstrap 5
